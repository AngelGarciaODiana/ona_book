Notes About This Book

Data

It is my intention to make all the data sets in the book available in convenient and self-contained packages in R and Python to coincide with the print release of this book in 2022. In the meantime, the datasets are all accessible via the URLs provided and you can always access them by following the code examples.

Accessibility

It is important to me to make as much of this book as possible accessible to vision-impaired readers. I will progressively be adding alt text to the figures in this book over the next weeks. I have also tried to make some of the most important visualizations more accessible to color blind readers by using a color blind palette wherever possible. This has been easier to do with visualizations produced in R with ggraph. My apologies if not all of the visualizations are color blind friendly and I would encourage any color blind readers to let me know1 if a particular visualization is problematic so I can try to address it.
